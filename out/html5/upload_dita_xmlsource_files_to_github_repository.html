Upload DITA XML source files to GitHub repository

Upload DITA XML source files to the repository to share them publicly.

  1. Go to http://www.github.com.
  2. Navigate to your public repository.
  3. Click Add file > Upload files.
  4. Drag and drop the entire folder to the repository, in the space provided.
    Note: Ensure that you upload the entire folder with all the supporting files like images etc., to the repository.
  5. Click Commit Changes.
