What are DITA maps?

Explains DITA maps at a high level.

DITA maps are XML documents that organize topics and other resources into structured collections of information. DITA maps specify hierarchy and the relationships among the topics. DITA maps typically have . ditamap file extensions.

DITA maps are key components for authoring DITA content. Managing these files and their referenced topics is an important feature of a DITA editor. The DITA Maps Manager in Oxygen enables you to view and edit DITA map files.
