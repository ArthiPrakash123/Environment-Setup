What are DITA Topics?

Briefly explains DITA topics.

What are Topics?

A topic is a unit of information with a title and content, short enough to be specific to a single subject or answer a single question, but long enough to make sense on its own and be authored as a unit. Topics are the basis for high-quality information.

Information typing is the practice of identifying types of topics that contain distinct kinds information, such as concepts, tasks, and reference information. Topics that answer different kinds of questions can be categorized under different information types. The base topic types provided by DITA (a generic topic, plus concept, task, and reference ) provide a starter set that can be adopted for immediate authoring.

What are the DITA Topic Types?

DITA includes three main topic types: Task, Concept, and Reference.

  • Tasks are used to describe how to perform a procedure, for example, to install a product.
  • Concepts present descriptive information to understand the background or context of a product.
  • Reference topics provide detailed facts such as product specifications, often in a table.

What is the structure of a topic?

The DITA standard defines how each topic type is structured. All topics have the same basic structure, regardless of the topic type.

Every topic contains some common elements, like Title, Prolog (for metadata like audience, category, keywords), Short Description, as well as some that are unique.
