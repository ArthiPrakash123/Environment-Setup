What is DITA?

Briefly describes DITA.

The Darwin Information Typing Architecture (DITA) specification defines a set of document types for authoring and organizing topic-oriented information. It is an open standard. that is defined and maintained by the OASIS DITA Technical Committee.

The name derives from the following components:

  • Darwin: because it uses the principles of specialization and inheritance, which is analogous to Charles Darwin's concept of evolutionary adaptation.
  • Information Typing: which means each topic has a defined primary objective and structure.
  • Architecture: DITA is an extensible set of structures.
