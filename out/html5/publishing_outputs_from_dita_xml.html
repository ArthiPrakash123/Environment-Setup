Publishing outputs from DITA XML

Use this procedure to publish outputs using the default transforms that come with the Oxygen XML authoring tool.

  1. Open the ditamap in the DITA Maps Manager.
  2. Click Configure Transformation Scenario(s).
  3. Select the following transforms:
    • DITA Map PDF - based on HTML5 & CSS (to publish to PDF)
    • DITA Map HTML5 (to publish to HTML5)
  4. Click Apply associated.
