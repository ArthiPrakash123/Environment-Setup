Creating a ditamap

Create a ditamap using Oxygen XML Author.

DITA maps are documents that organize topics and other resources into structured collections of information. DITA maps specify hierarchy and the relationships among the topics; they also provide the context in which keys are defined and resolved. DITA maps should have . ditamap file extensions. Use this procedure to create a ditamap using Oxygen XML Author.

  1. Open an instance of Oxygen XML Author.
  2. Frm the top left hand corner, click File > New
    The Choose a file template dialog box displays.
  3. Create a map as follows:
    1. In the field provided, type map.
    2. Choose Map [DITA / Topics]
    3. In the Title field, enter a name for your map.
    4. Observe the path on your computer where the file is being created. Ensure that the path is correct.
    5. Click Create.
