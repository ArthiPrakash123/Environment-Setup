Create a Public GitHub Respository

Create a personal GitHub account. See: Create a free Personal Github Account.

Use this procedure to create a public GitHub repository.

  1. On an instance of a web browser, enter www.github.com.
  2. On the top right corner of the page, click the drop-down arrow and select New repository.
    The Create a new repository page displays:
  3. Enter a project name in the Repository name field.
    For example, enter: My first DITA-XML project.
  4. Select Public to create a public repository.
    Important: Ensure that you create a public repository. That way anyone who has the URL to the repository will be able to access it.
  5. Optional: Enter a brief description in the field provided.
  6. Click Create repository.
